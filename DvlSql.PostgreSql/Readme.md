# DvlSql.PosgreSql

**DvlSql** is a powerful and easy-to-use library for creating `SELECT`, `UPDATE`, `INSERT`, and `DELETE` statements for PostgreSQL using a LINQ-like syntax.

---

## ✨ Features

* **Selects for PostgreSQL**
* **Insert statements for PostgreSQL**
* **Update statements for PostgreSQL**
* **Delete statements for PostgreSQL**

---

## 📦 Installation

You can install **DvlSql** via NuGet Package Manager:

```powershell
Install-Package DvlSql.PostgreSql -Version 0.1.0
```
